# 나만의 작은 냉장고
사용자가 자신의 냉장고 및 팬트리 내부의 식자재를 효과적으로 관리하고, 유통기한이 얼마 안남은 식재료들은 알림을 보내주고 그 식재료로 만들 수 있는 요리 레시피를 추천해주는 앱.

</br>
